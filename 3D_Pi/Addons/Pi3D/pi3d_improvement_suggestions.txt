Pi3D アドオン 改善・追加機能提案

対象: 3D_Pi / Addons / Pi3D
除外: パフォーマンス改善
観点: グラフィック、使い勝手、演出、UI

1. Fog 色 UI + Fog プリセット
- FogSettings.color と m_fogColor は既に存在するため、UI から色を変更できるようにする。
- RGB スライダーを追加する。
- プリセット例:
  - 朝霧
  - 夕焼け
  - 砂埃
  - 夜霧
  - 毒霧
- 実装コストに対して見た目の効果が大きい。

2. 雨の色・透明度 UI
- PiRain には alpha があるが、環境 UI から調整できない。
- 雨色も固定なので、雨の透明度・色・明るさを UI 化すると良い。
- 夜やホラー系プリセットで特に有効。

3. 地面モード追加
現在:
- Texture
- Gray

追加候補:
- None: 地面非表示
- White: ライティング確認用
- Dark: 夜・シルエット確認用
- Grid: スケール確認用
- Checker: UV / 遠近確認用
- Matte: ニュートラル床

実装済み:
- None
- White
- Grid

4. ライティングプリセット追加
現在:
- 朝
- 昼
- マジックアワー
- 夜

追加候補:
- 曇天
- 逆光
- 夕方
- 月夜
- ホラー
- 雪景色
- 砂漠

TOML で増やせる構造なので追加しやすい。

5. 効果チェイン UI のコンパクト化
- 現状は各段に全エフェクトのラジオリストが表示され、縦に長くなりやすい。
- 1 段 1 行表示にして、効果名クリックで選択リストを展開すると扱いやすい。

候補表示:
[ON/OFF] [効果名] [↑] [↓] [複製] [×]

6. 各エフェクトの個別リセット
- Bloom / Film Grain / DoF など、エフェクト単位で初期値に戻すボタンがあると試行錯誤しやすい。
- パラメータブロックに Reset ボタンを追加する案。

7. プリセットのタグ表示
- displayName と description に加えて tags を TOML に持たせる。
- 例:
  - cinematic
  - warm
  - dark
  - retro
  - anime
  - visibility
- UI で選択しやすくなる。

8. 現在の設定をプリセットとして保存
- 現在のエフェクトチェイン、環境、ライティングをユーザープリセットとして保存できると便利。
- 保存先候補:
  save/pi3d_user_presets.toml
- 機能候補:
  - 現在のエフェクトチェインを保存
  - 現在の環境設定を保存
  - 現在のライティング設定を保存
  - 名前を付けて保存
  - 削除

9. Sky / 背景表現の強化
- 現状はライティングプリセットの backgroundColor による単色背景。
- 追加候補:
  - 垂直グラデーション Sky
  - 夜空
  - 星
  - 月
  - 夕焼けグラデーション
  - 曇天背景
  - Fog 色と背景色の連動

10. Sun / 風向きの可視化
- シーン内または UI に方向を視覚化すると調整しやすい。
- 追加候補:
  - Sun 方向の矢印
  - 雨の風向き矢印
  - 小さい方位コンパス
  - ライト方向ミニビュー

11. 簡易カラーグレーディング
- Warm Grade より汎用的な色調整があると便利。
- 追加候補:
  - Exposure
  - Contrast
  - Saturation
  - Gamma
  - Temperature
  - Tint
  - Lift / Gamma / Gain 風の簡易調整

12. DoF のフォーカス補助
- DoF の調整は視覚的に難しいため、補助表示があると良い。
- 追加候補:
  - フォーカス距離ガイド
  - 画面中央の深度を取得してフォーカス
  - Focus Near / Far のデバッグ表示
  - 深度プレビュー表示

13. 天候プリセット
- 雨、Fog、Lighting、EffectChain をまとめて切り替えるプリセット。
- 例:
  - 晴れ
  - 曇り
  - 小雨
  - 豪雨
  - 霧
  - 夜雨
  - 砂埃
  - ホラー霧
  - マジックアワー

14. 雪エフェクト
- 雨の仕組みを流用して実装できそう。
- 調整項目:
  - 雪量
  - 落下速度
  - 横風
  - 粒サイズ
  - 透明度
  - 色

15. 砂埃 / パーティクル風オーバーレイ
- Dusty プリセットと相性が良い。
- 表現案:
  - 画面手前を漂う薄い粒
  - Fog 色を黄土色寄りにする
  - Film Grain と組み合わせる
  - 低速で横に流れる

実装優先度案:
1. Fog 色 UI + Fog プリセット
2. 雨の透明度 / 色 UI
3. 地面モード追加: None / White / Grid [実装済み]
4. ライティング TOML プリセット追加
5. 天候プリセット
6. 効果チェイン UI のコンパクト化
7. グラデーション Sky
8. 簡易カラーグレーディング
